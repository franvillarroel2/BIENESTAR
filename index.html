¿Qué es BIENESTAR?

BIENESTAR es un sistema inteligente de monitoreo para colmenas. Permite controlar el nivel de actividad sonora, la temperatura, humedad y posibles riesgos que afectan a las abejas.

Funcionalidades principales

  • 🎙️ Monitoreo sonoro de la colmena
  • 🌡️ Lectura de temperatura y humedad
  • 🌫️ Lectura de nivel de gas interno
  • 📡 Comunicación LoRa
  • 🌐 Lectura de datos en la web

Apoyando el cuidado de nuestras abejas 🐝

Un monitoreo constante puede marcar la diferencia entre una colmena saludable y una en riesgo.
